Interfacepublic interface IValidating extends IFeathersDisplayObject, IFeathersEventDispatcher
Implementors Scale3Image, Scale9Image, TiledImage

A display object that supports validation. Display objects of this type will delay updating after property changes until just before Starling renders the display list to avoid running redundant code.

Method Detail
validate()method
public function validate():void

Immediately validates the display object, if it is invalid. The validation system exists to postpone updating a display object after properties are changed until until the last possible moment the display object is rendered. This allows multiple properties to be changed at a time without requiring a full update every time.

A display object cannot validate if it does not have access to the stage.

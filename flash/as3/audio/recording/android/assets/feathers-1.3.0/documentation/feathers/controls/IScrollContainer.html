Packagefeathers.controls
Interfacepublic interface IScrollContainer extends IFeathersControl, IValidating, IFeathersDisplayObject
Implementors ScrollContainer

Defines functions for a Scroller subclass that delegates display list manipulations to its IViewPort.

Mainly intended for ScrollContainer, but other subclasses of Scroller could possibly implement it too.

See also
